# 🎨 Frontend Guide

The frontend is built with **React** and **Vite** for a fast and modern development experience.

## 📁 Directory Structure

- `src/components`: Reusable UI elements (e.g., `Navbar`).
- `src/pages`: Main application views (`Dashboard`, `Students`, etc.).
- `src/services`: Logic for interacting with backend APIs (`api.js`).
- `src/index.css`: Global styles and design system.

## 💎 Design System

We use **Vanilla CSS** with a premium aesthetic:
- **HSL Colors**: Controlled using CSS variables for consistency.
- **Glassmorphism**: Achieved with `backdrop-filter: blur()` and semi-transparent backgrounds.
- **Animations**: Subtle fade-in and slide-up effects to make the UI feel alive.

## 🛠️ Components Used
- **Lucide React**: For beautiful, consistent iconography.
- **React Router**: For client-side navigation between modules.
- **Axios**: For robust HTTP requests to the backend.
