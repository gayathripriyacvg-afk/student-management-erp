# 🎓 Student Management Mini ERP (MERN)

## 🔹 Goal

Build a system where you can **manage students, courses, and marks** with basic CRUD operations.

---

# 🧩 Core Modules (Keep it small)

## 1. Students Module (Main)

**Fields:**

* Name
* Email
* Age
* Course (reference or simple string)
* Enrollment Date

**Features (CRUD):**

* ➕ Add student
* 📄 View all students
* ✏️ Edit student
* ❌ Delete student

---

## 2. Courses Module (Optional but good)

**Fields:**

* Course Name
* Duration
* Instructor

**CRUD:**

* Add course
* View courses
* Update
* Delete

👉 You can store `courseId` in student

---

## 3. Marks Module (Simple version)

**Fields:**

* Student ID
* Subject
* Marks

**CRUD:**

* Add marks
* View marks per student
* Update
* Delete

---

# 🗄️ Database Design (MongoDB)

### Student Schema

```js
const StudentSchema = new mongoose.Schema({
  name: { type: String, required: true },
  email: String,
  age: Number,
  course: String, // or ObjectId if using Course collection
  enrolledAt: { type: Date, default: Date.now }
});
```

---

### Course Schema

```js
const CourseSchema = new mongoose.Schema({
  courseName: String,
  duration: String,
  instructor: String
});
```

---

### Marks Schema

```js
const MarksSchema = new mongoose.Schema({
  studentId: { type: mongoose.Schema.Types.ObjectId, ref: "Student" },
  subject: String,
  marks: Number
});
```

---

# 🔌 Backend API Design (Express)

## Student APIs

* `GET /students` → get all students
* `POST /students` → add student
* `PUT /students/:id` → update
* `DELETE /students/:id` → delete

---

## Course APIs

* `GET /courses`
* `POST /courses`
* `PUT /courses/:id`
* `DELETE /courses/:id`

---

## Marks APIs

* `GET /marks/:studentId` → get marks for student
* `POST /marks`
* `PUT /marks/:id`
* `DELETE /marks/:id`

---

# 🎨 Frontend (React)

## Pages

1. **Student List Page**

   * Table view
   * Buttons: Edit / Delete

2. **Add/Edit Student Form**

   * Input fields
   * Submit button

3. **Course Page** (optional)

4. **Marks Page**

   * Select student → show marks
   * Add marks form

---

# 🔁 Basic Flow

1. Add Course
2. Add Student (assign course)
3. Add Marks
4. View everything in tables

---

# 💡 Simple UI Features

* Search student by name
* Filter by course
* Show total marks per student
