# 🎓 Student Management Mini ERP

A premium MERN stack application for managing students, courses, and academic performance.

## 🚀 Features

- **Students Module**: Full CRUD operations for managing student records.
- **Courses Module**: Manage educational courses and instructors.
- **Marks Module**: Track academic performance per student and subject.
- **Premium UI**: Dark-themed design with glassmorphism, HSL colors, and smooth transitions.
- **Responsive Layout**: Designed for clarity and ease of use.

## 🛠️ Tech Stack

- **Frontend**: React (Vite), Vanilla CSS, Lucide React, Axios.
- **Backend**: Node.js, Express, MongoDB, Mongoose.
- **Design**: Premium HSL-based design system with glassmorphism.

## 📦 Getting Started

### Prerequisites

- Node.js installed.
- MongoDB running locally or a MongoDB Atlas URI.

### Setup

1. **Clone the repository**
   ```bash
   git clone <repo-url>
   cd student-management-erp
   ```

2. **Backend Setup**
   ```bash
   cd backend
   npm install
   # Create a .env file with PORT and MONGO_URI
   npm start
   ```

3. **Frontend Setup**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

## 🎨 Design Philosophy

Built with visual excellence in mind, using:
- **HSL Colors**: Curated palettes for a modern look.
- **Glassmorphism**: Subtle blur effects and semi-transparent layers.
- **Micro-animations**: Smooth transitions for an alive interface.
